# Chức năng Order và Checkout

Base path: /api/orders/

## User endpoint
- GET /api/orders/
- POST /api/orders/checkout/
- GET /api/orders/{id}/

## Admin endpoint
- GET /api/orders/admin/
- GET /api/orders/admin/{id}/
- PATCH /api/orders/admin/{id}/

## Mục đích
- Tạo đơn hàng từ cart, quản lý trạng thái đơn, và xem lịch sử đơn.
